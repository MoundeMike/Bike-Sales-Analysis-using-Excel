# Bike-Sales-Analysis-using-Excel
This project analyzes bike sales data using Microsoft Excel to identify sales trends, customer segments, and regional performance.
The analysis involved data cleaning, pivot table analysis, and building an interactive dashboard.

Analysis Performed
- Cleaned raw sales data (handled missing values, standardized categories)
- Created new features(Age Bracket)
- Built pivot tables to analyze:
  - Sales by gender
  - Sales by customer age group
  - Sales by commute distance
- Designed an interactive Excel dashboard for insights
